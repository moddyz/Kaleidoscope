# Kaleidoscope

This is an implementation of the Kaleidoscope language, following the tutorial here:
https://llvm.org/docs/tutorial/MyFirstLanguageFrontend

## Building

Quick build:
```bash
./build.sh <INSTALLATION_PATH>
```

Requirements:
- \>= CMake-3.10
- \>= llvm-9.0.1
