Detailed Description

CodeGenContext is a structure storing the internal state of the generated IR code, and various LLVM objects which contribute to code-generation.

This class has ownership of the following objects:

  • LLVM context
  • a module.
  • an IR builder.
  • a function pass manager.

The module and function pass manager are not initialized upon CodeGenContext construction, Use InitializeModule() in situations which only demand IR and/or object code generation. Or, use InitializeModuleWithJIT() in situations which require both IR code generation and JIT execution.

MoveModule() facilitates ownership transfer of its module over to the JIT engine.
